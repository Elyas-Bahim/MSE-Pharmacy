# 💊 Pharmacy Management Website

A responsive pharmacy website built using **HTML** and **CSS** as a university project.

## 📖 About

Pharmacy Management Website is a front-end project developed as part of a university assignment. The project focuses on designing a clean and user-friendly interface for a pharmacy, allowing users to view pharmacy information, products, and services through a simple and organized layout.

## ✨ Features

* Responsive design
* Clean and modern user interface
* Pharmacy product showcase
* Medicine categories section
* Services information
* About pharmacy section
* Contact section
* User-friendly navigation

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Elyas-Bahim/MSE-Pharmacy.git
```

2. Open the project folder.

3. Run the `msePharmacy.html` file in your web browser.

## 🎯 Purpose

This project was created as a university project to practice front-end web development skills and apply HTML and CSS concepts in building a real-world pharmacy website interface.

## 📱 Responsive Design

The website is designed to work on:

* Desktop
* Tablet
* Mobile
